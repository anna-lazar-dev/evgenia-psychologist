# 🧠 Сайт для психолога

Одностраничный сайт для частной практики психолога с формой записи и адаптивным дизайном.

## 🔗 Демо
https://sahatskaya.ru/

## 📌 Задача
Создать аккуратный и спокойный сайт для специалиста, который:
- вызывает доверие
- удобно читается с телефона
- позволяет быстро записаться на консультацию

## ⚙️ Реализация
- Адаптивная верстка (HTML, CSS)
- Интерактивные элементы и анимации
- Форма обратной связи через Formspree
- Подключение собственного домена
- Деплой через GitHub Pages

## 📱 Особенности
- Чистый минималистичный дизайн
- Удобная навигация
- Оптимизация под мобильные устройства
- Быстрая загрузка

## 🛠️ Стек
HTML / CSS / JavaScript / GitHub Pages / Formspree
